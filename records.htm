Club Records

Initially autogenerated from PowerOf10 club page and runbritain rankings 2003 - 2023 on 2023-07-09.

Outputting maximum 10 places overall per event and 3 per age group.

The idea is to then fill historical gaps or correct any inadmissable performances manually, or by adding an additional input file to this automated analysis so that it can be rerun incorporating the information unknown to powerof10 to keep things up to date. While this was done for road running, similarly the T&F records it finds could be merged with the official club T&F records manually or by absorbing them from an input file.

See https://github.com/charlie-wartnaby/powerof10-tools for source code.
